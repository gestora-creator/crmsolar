useGruposEconomicos.ts + useVinculos.ts\nImpacto:      Erro 23505 (conflict), inconsistência de dados\nEsforço:      4-5 horas\nSolução:      RPC atômico no Supabase\n\nBefore:                    After:\nUser A: SELECT            DB: /* atômico */\nUser B: SELECT            - Try insert\n  ↓                       - On conflict: ignore\nAmbos INSERT              - Retorna resposta única\n  ↓\nConflict!\nErrora UX ❌              Sucesso garantido ✅\n```\n\n---\n\n### 3. 🔄 Invalidação Cascata\n```\nProblema:     invalidateQueries(['clientes']) invalida TUDO\nArquivos:     useClientes.ts + useContatos.ts + useVinculos.ts\nImpacto:      15-30 refetches simultâneos\nEsforço:      3-4 horas\nSolução:      Query Key Factory + invalidação específica\n\nBefore:                    After:\nCreate 1 cliente           Create 1 cliente\n  ↓                          ↓  \nInvalidate ['clientes']    Invalidate apenas\n  ↓                       ['clientes', 'list']\n15+ queries refetch     1-2 refetches máximo\n  ↓\n200-300ms freezeUI      50-100ms operação\nNasty UX ❌              Smooth UX ✅\n```\n\n---\n\n### 4. 🔑 Cardinalidade Alta\n```\nProblema:     500+ cache keys simultâneos\nArquivos:     useClientes.ts, useContatos.ts\nImpacto:      1-5 MB memória por usuário\nEsforço:      1-2 horas\nSolução:      Remover pageSize, aumentar staleTime\n\nBefore:                    After:\n['clientes',              ['clientes', 'list',\n  '',                       { searchTerm: '', page: 0 }\n  0,                      ]  \n  30                       \n]                         (Menos keys, mas\n                           mais stale time)\n\n500 keys × 3KB = 1.5 MB   500 keys × 3KB = 1.5 MB\n+ Refetch frequente       + Menos refetches\n= Memória desperdiçada    = Eficiência otimizada\n```\n\n---\n\n### 5. 📍 Sem ETags\n```\nProblema:     Sem validação eficiente (ETag/Last-Modified)\nArquivos:     Todos os endpoints\nImpacto:      Cache não aproveita validação 304\nEsforço:      2-3 horas\nSolução:      Adicionar suporte ETag\n\nBefore:                    After:\nGET /api/clientes         GET /api/clientes\n  ↓                         (If-None-Match: etag)\n200 OK + 50KB             ↓\n(sempre transfere)        304 Not Modified\n                          (Transfere 0 bytes)\n```\n\n---\n\n## 📊 IMPACTO
